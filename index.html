Introduction

Vicy is a cute and friendly Discord bot designed to help and entertain users. Your privacy is very important to us. This policy explains how we handle the information collected by Vicy.

Data Collection

  • Vicy does not request or collect any sensitive personal information from users.
  • For server features, we only collect your Discord username and ID, which is displayed within a guild for leaderboard or fun interactions.
  • TikTok Integration: To provide LIVE notifications, Vicy accesses public TikTok usernames and LIVE status via official TikTok APIs.

Data Usage

Collected Discord usernames are used only to show activity, leaderboard stats, and interactions inside the server. Data retrieved from TikTok is used strictly to trigger real-time notifications in authorized Discord channels. Vicy does not share, sell, or distribute any user information outside the server.

Data Retention

Usernames and LIVE statuses are stored temporarily for bot functionality within the server and are not stored permanently outside the bot system.

Third Parties

Vicy interacts with TikTok APIs solely for the purpose of the notification feature. Aside from this official integration, Vicy does not share user data with any third party. Data is only visible inside the server where the bot is used.

Changes to Privacy Policy

We may update this Privacy Policy occasionally. Users will be informed of major changes via bot announcements or server messages.
